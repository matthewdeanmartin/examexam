# examexam docs

`examexam` has four main audiences, and this site is organized around them.

- If you want to study with an existing question bank, start with [Taking Tests](usage/taking-tests.md).
- If you want to generate or validate question banks with an LLM, start with [Creating Tests](usage/generating-tests.md).
- If you want to understand how the codebase works, start with [Hacking On Examexam](extending/how-it-works.md).
- If you want the general contributor workflow, start with [Contributing](extending/CONTRIBUTING.md).

The short version: `examexam` is a local-first tool that stores question banks as TOML, uses provider APIs to generate and validate content, and lets you take exams through either a Rich terminal UI or a localhost web UI.
